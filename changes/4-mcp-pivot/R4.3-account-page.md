# R4.3 — Account page: MCP token generation

**Status:** done
**Files:** `src/app/(app)/account/`, `src/lib/actions/account.ts`, `prisma/schema.prisma`

## Spec

`/account` — accessible to all ACTIVE users via a sidebar "Account" link.

- Shows current token status (masked if set, "No token" if not).
- "Generate token" / "Regenerate token" client action → `generateMcpToken()` server action.
- Token shown once after generation with a copy button (client state, not URL param).
- MCP client config block: a JSON snippet to paste into Claude Desktop / Cursor MCP config.

**DB change:**
```sql
ALTER TABLE "User" ADD COLUMN IF NOT EXISTS "mcpToken" TEXT;
CREATE UNIQUE INDEX IF NOT EXISTS "User_mcpToken_key" ON "User"("mcpToken");
```
Applied via `scripts/apply-schema.ts`.

### MCP client setup (for end users)
Paste into `claude_desktop_config.json` or Cursor MCP settings (replace `localhost:3000` with the deployed URL when live):
```json
{
  "mcpServers": {
    "seed-tracker": {
      "url": "http://localhost:3000/api/mcp",
      "headers": { "Authorization": "Bearer <your-token>" }
    }
  }
}
```
