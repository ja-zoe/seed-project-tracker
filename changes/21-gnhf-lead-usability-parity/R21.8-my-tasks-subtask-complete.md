# R21.8 — One-click subtask complete on My Tasks

**Status:** done (merged in `842c620`)
**Files:** `src/app/(app)/my-tasks/page.tsx`, `src/lib/actions/deliverables.ts`

## Spec

**Problem:** On My Tasks, action items had a quick-complete control but subtasks were an inert whole-card
link — completing a subtask forced a multi-step detour through the project page.

**Approach:** Subtask rows expose a one-click "Mark complete" button calling `updateSubtaskStatus`, matching
the action-item control. The card was restructured so the project link is scoped to the title/meta only,
freeing the row for the form button without nesting interactive elements. `updateSubtaskStatus` now also
revalidates `/my-tasks` so the completed subtask leaves the open-work list immediately.

## Tests

Retroactive — verified during the run: typecheck + completion removes the row on My Tasks. No dedicated
Playwright spec.
