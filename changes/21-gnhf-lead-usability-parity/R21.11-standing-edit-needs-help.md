# R21.11 — Standing edit modal lacked the needs-help toggle

**Status:** done (merged in `4b59a9f`)
**Files:** `src/components/status-update-controls.tsx`

## Spec

**Problem:** The project-standing submit form has an "I need help from the PM" toggle, but the edit modal in
`StatusUpdateControls` did not — a lead couldn't add or clear the PM-help escalation when editing a
standing. The modal's `submit()` already serialized `needsHelp`/`helpNeeded` from initial values and the
backend `updateStatusUpdate` handled both fields; nothing let the user change them.

**Approach:** Add the checkbox plus a conditional help-description textarea to the edit modal, wired to the
existing `needsHelp`/`helpNeeded` state. Pure missing-control fix — no data-flow or backend change.

## Tests

Retroactive — verified during the run: typecheck + toggling/clearing help via the edit modal persists. No
dedicated Playwright spec.
