# R5.1 — Notification modal positioning fix

**Status:** done
**Files:** `src/components/notification-bell.tsx` (line ~94)

## Spec

**Problem:** The notification popup clips off the left edge of the screen. It is `absolute bottom-full right-0`. The bell sits at the bottom-right corner of the left sidebar (~212px from the viewport's left edge). `right-0` aligns the popup's right edge with the bell button's right edge; because the popup is `w-80` (320px), it extends 320px **leftward** from that anchor, running ~108px off-screen.

**Approach:** Change `right-0` → `left-0`. This anchors the popup's **left** edge to the bell container's left edge and extends it rightward into the main content area, where it's fully visible.

```diff
- className="absolute bottom-full right-0 mb-2 w-80 ..."
+ className="absolute bottom-full left-0 mb-2 w-80 ..."
```

No other positioning changes needed.
