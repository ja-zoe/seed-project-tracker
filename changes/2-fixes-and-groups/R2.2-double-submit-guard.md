# R2.2 — Prevent double-submit on create forms

**Status:** done
**Files:** `src/components/submit-button.tsx` (new) + all create/edit forms

## Spec

**Problem:** Subtask / action-item / deliverable create forms could be submitted twice.

**Approach:** Add a `SubmitButton` component using `useFormStatus` that disables itself while the form is pending. Use it on all create/edit forms.
