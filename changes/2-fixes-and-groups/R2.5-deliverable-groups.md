# R2.5 — Deliverable groups

**Status:** done
**Files:** `prisma/schema.prisma`, `src/app/(app)/projects/[id]/` detail + timeline, deliverable create/edit forms

## Spec

**Problem:** Deliverables had no grouping; long lists were flat.

**Approach:** Add an optional `group` label to each deliverable. Visually separate groups (group header per section) in the project detail and timeline views; add a group input to create/edit forms.

**DB change:**
```sql
ALTER TABLE "Deliverable" ADD COLUMN IF NOT EXISTS "group" TEXT;
```
Applied via `scripts/apply-schema.ts`. `prisma/schema.prisma`: `group String?` on `Deliverable`.

## Notes / log

- 2026-06-26 — Nullable `group` field added. Project detail shows grouped sections with a group header; create/edit forms include a group input.
