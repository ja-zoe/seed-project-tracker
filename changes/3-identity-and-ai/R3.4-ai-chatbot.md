## Spec (as originally shipped)

**Goal:** A role-scoped AI assistant that performs work in the dashboard on behalf of the logged-in user (create action items, draft status updates, build timelines, assign subtasks), constrained by that user's actual permissions. Floating chat window, bottom-right of every app page.

**Cost model:** AI cost falls entirely on the individual user — no platform-level API keys. Users connect their own providers via OAuth (where supported) or API key, and can switch providers/models from a dropdown.

### Protocol: MCP as the tool-calling layer
Tool use exposed via the Model Context Protocol (provider-agnostic). SEED tools defined once as an MCP server; the Vercel AI SDK's `experimental_createMCPClient` connects each provider. Adding a provider needs no tool-definition changes; switching models mid-conversation is safe; a user's own external MCP server can be layered in.

Every provider card showed both an OAuth button and an API-key input (OAuth disabled with "coming soon" tooltip where unsupported) so no rework was needed when OAuth ships.

### UI
`ChatWidget` ("use client"): `fixed bottom-6 right-6 z-50`. Collapsed = circular `ChatCircleDots` FAB (forest-green). Expanded = ~380×540 panel with header (label, active model chip, replay-tour, settings, minimize), scrollable message list, and a textarea footer (Enter sends, Shift+Enter newline). Open state persisted in `localStorage`. Connection panel with Connect / My-connections tabs. Three-step onboarding overlay on first open.

### Backend
- `src/app/api/chat/route.ts` — POST, session-required, streaming. Built system prompt from role/permissions/project context; called `streamText({ model, system, messages, tools, maxSteps: 5 })`.
- `src/app/api/mcp/route.ts` — SEED MCP tool server. Tools: `createActionItem`, `createStatusUpdate`, `listProjects`, `getProjectDetail`, `createDeliverable`, `createSubtask`, `updateProjectStatus` — each RBAC-gated.
- Optional encrypted credential storage (`UserAiProvider` table, AES-256-GCM via `src/lib/crypto.ts`, gated by `AI_KEY_ENCRYPTION_SECRET`). **Never applied** — shipped session-only.

### Packages added (later removed in set 4)
`ai`, `@ai-sdk/anthropic`, `@ai-sdk/openai`, `@ai-sdk/google`, `@ai-sdk/react`.
